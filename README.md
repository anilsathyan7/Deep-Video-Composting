# Deep-Video-Composting


## Dependencies

* Python 2.7
* Tensorflow
* Caffe
* Opencv, PIL, Scikit-Image

## Features

* A paint canvas with draw & erase.
* Recognizes 100 different objects/shapes.
* Shows top three predictions.
* Provides voice output with tts.
